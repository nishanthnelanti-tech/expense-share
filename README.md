# BillBuddies
BillBuddies is a bill spliting application to streamline group bill settlemen, using MERN stack

## How to use

```
   
Use node command to start server side server
1. cd/server
   node app.js
   
Use npm start to start the client side sever . (In another Terminal)
2. cd/client
   npm start
```

And point your browser to `http://localhost:3000`.
